Для сборки проекта необходим c++20

Инструкция по запуску:
    Без флагов по умолчанию запускается сканирование текущей директории (build) в .media_files домашней директории

    Флаги:
        --network - Включает использование localhost для публикации результатов сканирования
        --interval T - Устанавливает интервал между сканированиями равным T миллисекунд
        --path P - Сканирует директорию P
